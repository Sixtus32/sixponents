# `<sx-theme-selector>`

## Instalación

```sh
npm install sx-theme-selector
```

```sh
yarn add sx-theme-selector
```

> NOTA: `sx-theme-selector` se distribuye como un Módulo de JavaScript ES2020 y utiliza la API de Elementos Personalizados. Es compatible con todos los navegadores modernos, incluyendo Chrome, Firefox, Safari, Edge e IE11, pero se requiere un paso adicional de herramientas para resolver los _bare module specifiers_, así como la transpilación y los polyfills para Edge e IE11.
